10 Common Linux Commands

Name of Command Brief Description An Exmaple of Common Usage
1. mkdir The mkdir command allows users to create directories or folders. mkdir [options...] [directories ...]
2. cd The cd command is used to change the current working directory. cd [OPTIONS] directory
3. ls The ls command is used to list information about files and directories within the file system. ls [OPTIONS] [FILES]
4. mv The mv command is used to rename and move and files and directories from one location to another. mv [OPTIONS] SOURCE DESTINATION
5. cat The cat command is used to display the contents of one or multiple text files, combine files by appending the contents of one file to the end of another file, and create new files. cat [OPTIONS] [FILE_NAMES]
6. echo The echo command is used in shell scripts to display a message or output the results of other commands. echo [-neE] [ARGUMENTS]
7. kill The kill command is used to send a signal to specified processes or process groups, causing them to act according to the signal. kill [OPTIONS] [PID]...
8. clear The clear command is used for clearing the terminal. clear
9. scp The scp command is used to securely copy files and directories between two locations. scp [OPTION] [user@]SRC_HOST:]file1 [user@]DEST_HOST:]file2
10. rm The rm command is used for removing files and directories. rm [OPTIONS]... FILE...
